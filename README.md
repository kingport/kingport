- **主导公司内部 UI 组件库建设**
  - 从 0 到 1 起草并研发公司内部通用 UI 组件库，结合公司品牌设计规范，统一视觉与交互标准
  - 推动组件库在多个内部系统中复用落地，提升系统一致性、开发效率与维护性
  - 将原先偏项目制的页面开发模式逐步沉淀为可复用的前端基础能力

- **主导试算组件 SDK 平台化建设**
  - 负责统一试算能力的抽象、设计与研发，将原本分散在各业务线中的重复试算逻辑收敛为标准化 SDK
  - 建立统一接入方式，降低各业务线重复开发成本，减少逻辑分散带来的维护负担
  - 提升试算能力在多业务场景下的接入效率与一致性，为后续扩展与治理打下基础

- **参与核心交易系统建设与持续优化**
  - 面向核心交易链路，参与复杂业务模块开发与交互实现
  - 持续推动前端能力由“单点开发”向“通用复用、标准沉淀、平台协同”演进
  - 在业务快速迭代背景下平衡交付效率、代码质量与长期维护成本
